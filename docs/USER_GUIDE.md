## Working with Roadmaps

### Understanding Roadmaps

A **roadmap** is your personalized learning path containing:
- **Steps (Nodes)**: Individual learning topics
- **Dependencies**: Prerequisites between topics
- **Assessments**: Tests to prove mastery
- **Resources**: Additional learning materials

### Creating a New Roadmap

#### Method 1: AI-Generated Roadmap

1. Click **"Create Roadmap"** button
2. Fill in the form:
   - **Learning Goal**: "I want to learn..."
   - **Subject Area**: Technology, Business, Language, etc.
   - **Target Level**: Where you want to reach
   - **Time Commitment**: Hours per week
   - **Additional Context**: Any specific focus areas
3. Click **"Generate Roadmap"**
4. Wait 10-30 seconds for AI to create your path
5. Review and **"Activate"** the roadmap

**Example:**
```
Learning Goal: Build full-stack web applications
Subject Area: Web Development
Target Level: Job-ready professional
Time Commitment: 15 hours/week
Additional Context: Focus on React and Node.js, 
want to build a personal portfolio site
```

#### Method 2: Template-Based Roadmap (Coming Soon)

Choose from pre-made templates for common goals:
- "Frontend Developer Career Path"
- "Data Science Fundamentals"
- "JavaScript Mastery"

### Viewing Roadmap Details

1. Go to **Roadmaps** tab
2. Click on any roadmap card

**Roadmap Page Shows:**
- **Header**: Title, description, progress
- **Visual Map**: Interactive node diagram
- **Step List**: All topics in order
- **Statistics**: Completion rate, time spent

### Navigating the Visual Roadmap

**Node Colors:**
- ğŸŸ¢ **Green**: Completed
- ğŸ”µ **Blue**: Available (unlocked)
- ğŸŸ¡ **Yellow**: In progress
- ğŸ”´ **Red**: Needs review (failed assessment)
- âšª **Gray**: Locked (prerequisites not met)

**Interacting with Nodes:**
- **Click** a node to view details
- **Drag** nodes to reposition (visual only)
- **Hover** to see quick info
- **Double-click** to start learning

### Editing Roadmaps

#### Reorder Steps
1. Open roadmap
2. Click **"Edit Mode"**
3. Drag and drop nodes to reorder
4. Dependencies auto-update
5. Click **"Save Changes"**

#### Add Custom Steps
1. Open roadmap
2. Click **"+ Add Step"**
3. Fill in:
   - Title
   - Description
   - Estimated time
   - Difficulty
   - Prerequisites (select from existing nodes)
4. Click **"Add Step"**

#### Delete Steps
1. Open roadmap in **Edit Mode**
2. Click the **"ğŸ—‘ï¸"** icon on a node
3. Confirm deletion
4. Dependent nodes will be updated

> âš ï¸ **Warning**: Deleting a step with many dependencies may affect your roadmap structure.

### Taking Notes

1. Scroll to **"My Notes"** section
2. Type your reflections and key takeaways
3. Notes auto-save every 10 seconds
4. Access notes anytime from lesson page

### Taking an Assessment

When you're ready to prove mastery:

1. Click **"Take Assessment"** button
2. Review assessment info:
   - Number of questions
   - Time limit (if any)
   - Passing score (usually 70%)
   - Number of attempts allowed
3. Click **"Begin Assessment"**

**During Assessment:**
- Answer all questions
- Use **"Previous"** and **"Next"** to navigate
- Click **"Submit"** when done
- Can't go back after submission

**After Assessment:**
- View your **score** immediately
- See **correct/incorrect answers**
- Read **detailed explanations**
- If passed (â‰¥70%): Next nodes unlock! ğŸ‰
- If failed (<70%): Review feedback and retry

### Retrying Failed Assessments

1. Study the feedback carefully
2. Review lesson content
3. Use additional resources
4. Ask AI tutor for clarification
5. Click **"Retry Assessment"**

**Attempt Limits:**
- **First 3 attempts**: Unlimited
- **After 3 failures**: Must wait 24 hours OR contact tutor

## Troubleshooting

### Can't access a lesson node?

**Solution:**
- Check if **prerequisites are completed**
- Verify node is **unlocked** (blue, not gray)
- Refresh the page
- Clear browser cache

### AI tutor not responding?

**Solution:**
1. Check your internet connection
2. Refresh the chat page
3. Try typing instead of voice (or vice versa)
4. Contact support if issue persists

### Lost my streak but I logged in!

**Solution:**
- **Logging in â‰  learning activity**
- You must complete a lesson, quiz, or spend 15+ minutes studying
- Streaks reset at midnight in your timezone
- Contact support if you believe it's an error

### Quiz won't submit?

**Solution:**
- Answer ALL questions
- Check for unanswered required questions
- Look for validation errors (red highlights)
- Try a different browser
- Contact support with quiz ID
